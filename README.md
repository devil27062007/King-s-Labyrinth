## King's Labyrinth

Kings's Labyrinth is a small 2D platform like a maze solving game where you guide a tiny king through castle rooms , dodge blocks and pass through doors to reach the next Place.

There are three rooms if you crossed three rooms you win.
The goal of this project was simple : build something that feels playful , retro and satisfying

## Controls
- `A` -> move left
- `D` -> move right
- `W` -> jump and enter the room
